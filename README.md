# AlvaroJimenez - Portfolio

Personal portfolio website for Alvaro Emmanuel Jimenez — Mechatronics Engineer, Product Designer & Systems Architect.

**Live site:** [alvaroemmanueljo.github.io](https://alvaroemmanueljo.github.io)

## Structure

```
├── index.html           → Main landing page
├── about.html           → About me
├── projects.html        → Portfolio grid
├── achievements.html    → Awards & recognitions
├── contact.html         → Contact form
├── sketches.html        → 3D prints gallery
├── racing.html          → Racing projects
├── ideas.html           → Product ideas
├── single-project.html  → Project detail template
├── projects/            → Individual project pages
├── css/                 → Stylesheets (Bootstrap 3, plugins)
├── js/                  → Scripts (jQuery, Isotope, OWL Carousel)
├── fonts/               → Web fonts (Azedo, icon fonts)
├── images/ & img/       → Site images
├── First9/              → Project media (first 9 projects)
├── 1. Next 4/           → Project media (next 4 projects)
├── ajax/                → AJAX portfolio templates
├── subweb/              → Secondary theme pages
├── assets/              → Additional assets
└── tools/               → Utility scripts
```

## Tech Stack

- HTML5 + Bootstrap 3
- jQuery 1.11.2
- Isotope.js (masonry grid + filtering)
- OWL Carousel (sliders)
- Magnific Popup (lightbox)
- WOW.js + Animate.css (scroll animations)

## Tools

- `tools/update_projects.py` — Bulk update project pages
- `tools/update_projects.bat` — Run project updates (Windows)
- `tools/update_colors.bat` — Bulk color replacement across pages
